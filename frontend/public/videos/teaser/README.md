# Dossier Vidéos Teaser

Vous pouvez placer vos propres vidéos ici pour qu'elles soient lues par l'application.

## Mode d'emploi
1. Placez votre fichier vidéo (ex: `teaser.mp4`) dans ce dossier : `frontend/public/videos/teaser/`.
2. Faites un **Git Push** vers Vercel / Render.
3. Dans votre panneau Admin, laissez le champ "URL de la Vidéo Teaser" vide pour utiliser `/videos/teaser/teaser.mp4` par défaut.
4. Si vous avez nommé votre fichier différemment (ex: `promo.mp4`), tapez simplement `/videos/teaser/promo.mp4` dans le champ de l'Admin.

## Recommandations
- **Format** : Utilisez le format `.mp4`.
- **Poids** : Essayez de garder la vidéo sous les 5-10 Mo pour un chargement instantané chez les utilisateurs.
- **Compression** : Utilisez un outil comme Handbrake ou Adobe Premiere pour compresser la vidéo en H.264.
